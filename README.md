# Hub de Atalhos

Organizador de links com iframe persistente e interface dark mode.

## 🚀 Funcionalidades

- Organize links por grupos
- Iframes persistentes (mantêm estado ao alternar)
- Interface dark/glass
- CRUD completo de grupos e links
- Salva dados no localStorage

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
